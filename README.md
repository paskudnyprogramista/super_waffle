## Prerequisites

### Dependencies

To install dependencies:

`$ bundle install`

### Database

To setup you database:

`$ bundle exec rails db:setup`

## Server

To start your app:

`$ bundle exec rails s`

## Test

Te test your app:

`$ bundle exec rspec`
